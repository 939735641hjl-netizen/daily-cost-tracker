# 日均费用追踪器

一个轻量的纯前端每日支出追踪工具，无需后端、无需注册，数据存储在本地浏览器。

## 功能

- **汇总卡片**：今日支出 / 本月日均 / 本月总计 / 记录天数
- **添加支出**：日期、金额、8 种分类（餐饮/交通/购物/娱乐/医疗/居家/教育/其他）、备注
- **明细列表**：按日期分组，支持按月份 + 分类双重筛选，可单条删除
- **分类汇总**：本月各分类占比横向进度条
- **数据持久化**：`localStorage`，刷新/关闭页面不丢失

## 使用

直接用浏览器打开 `index.html` 即可，无需构建步骤。

```
open index.html
```

```
daily-cost-tracker/

```
