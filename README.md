# 日均费用

模仿有数 App 资产摊销功能的单页工具。记录电子产品等贵重物品，自动计算每天分摊多少钱——用得越久越便宜，帮你判断东西值不值、该不该换。

**在线访问：** https://939735641hjl-netizen.github.io/daily-cost-tracker/

## 功能

- **总览看板**：今日总日均 / 追踪件数 / 日均最贵物品
- **10 种品类**：手机/平板、电脑、相机、游戏、音频、家电、交通、穿戴、书/课程、其他
- **颜色分级**：< ¥10/天 绿色·¥10~50 黄色·> ¥50 红色
- **使用进度条**：填写预计年限后显示已用百分比
- **底部抽屉**：添加/编辑表单、点卡片看详情、说明页
- **三种排序**：日均↓ / 日均↑ / 日期↓
- **数据持久化**：`localStorage`，无需注册、无后端

## 核心公式

```
日均 = 购入价格 ÷ max(今天 − 购买日期, 1)
本月 = 日均 × 30　　本年 = 日均 × 365
```
